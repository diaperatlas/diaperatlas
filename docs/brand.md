# BRAND

## Brand Identity

Name

DiaperAtlas

Tagline

ESPLORA. CONFRONTA. SCEGLI.

Primary Positioning

Il punto di riferimento italiano per scegliere il pannolone giusto.

---

# Brand Purpose

DiaperAtlas aiuta le persone a prendere decisioni migliori.

Non vende un prodotto.

Riduce l'incertezza prima dell'acquisto.

L'obiettivo non è convincere.

L'obiettivo è chiarire.

---

# Brand Personality

DiaperAtlas deve essere percepito come:

• indipendente

• competente

• trasparente

• calmo

• essenziale

• affidabile

• metodico

Mai come:

• aggressivo

• provocatorio

• sensazionalistico

• commerciale

• infantile

• medicale

# Visual Philosophy

The interface should feel like reading an editorial publication.

Not an ecommerce.

Not a startup.

Not a corporate website.

The experience should communicate trust before functionality.

# Logo

Official logo

DiaperAtlas symbol + wordmark

Blue drop inside the "A"

The logo should never be redesigned unless there is a strategic reason.

---

# Colors

Primary

#0B1D33

Accent

#2E7BCF

Background

Warm white

Neutral gray

Use color sparingly.

Blue exists to guide attention.

Not to decorate.

---

# Typography

Headings

Editorial serif

Body

Modern sans serif

Typography is the primary visual element.

Not illustrations.

---

# Photography

When photography is used:

Real products

Neutral backgrounds

Soft shadows

No exaggerated lighting

No artificial lifestyle scenes

No stock photos with smiling models.

---

# Illustration

Illustrations should be minimal.

Never decorative.

Every illustration must explain something.

If an illustration does not improve understanding, remove it.

---

# Icons

Simple outline icons.

Consistent stroke.

Never colorful.

Never playful.

---

# Motion

Minimal.

Purposeful.

No animation exists only because it looks nice.

---

# UX Principles

Less scrolling.

More clarity.

Fewer words.

Better hierarchy.

Every section must answer one question.

Never create sections only to make the page longer.

---

# Brand Promise

Visitors should leave the website thinking:

"I trust this project."

Not

"I want to buy something."

Trust always comes before commerce.

---

# Final Rule

Whenever a design decision is uncertain, choose the calmer option.

DiaperAtlas should age well.

Not chase design trends.


---

We will never use fake urgency.

We will never exaggerate product performance.

We will never hide affiliate relationships.

We will never publish sponsored reviews disguised as independent opinions.

We will never prioritize monetization over trust.
